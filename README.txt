******************************************************
******  	VIDEO TRACKING EVALUATOR       *******
******************************************************

**** Pre-requisits:
Make sure to have the following python libraries installed:
-wget
-shapely
-PyQt5
-pylatex
-pyhtgen
-opencv

Add trackers results and sequences in their corresponding folders
Make sure to follow the VOT format of both fields:
http://www.votchallenge.net/

**** How to run:
In console go to the path of the main.py file and run:

python main.py

**** Results are visible in the file:
evaluation_results.html
